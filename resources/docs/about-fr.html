Darwin Core Archive (DWC-A) est un standard de données informatique de la biodiversité qui fait de l'utilisation des termes Darwin Core pour produire un seul ensemble de données autonome pour la présence de l'espèce ou taxinomique (espèces) de données. Il est le format préféré pour la publication de données de la Global Biodiversity Information Facility. Vous exportez vos données comme un ensemble d'une ou plusieurs texte (CSV). Un descripteur de fichier XML simple (appelé meta.xml) est nécessaire pour informer les autres comment vos fichiers sont organisés.

Cette application Web génère le descripteur de fichier XML appelé "meta.xml" (voir l'onglet suivant dans l'interface utilisateur) pour vous.

Le processus est simple. Vous devez sélectionner les termes correspondant à vos données. Cet outil aide en fournissant une liste de termes et d'explications (sur la souris hover). Pour utiliser cet utilitaire vous:

1. Sélectionnez les conditions standard et les extensions nécessaires pour cartographier les données sur la biodiversité que vous souhaitez partager dans votre archive en ouvrant les dossiers à gauche et à la sélection des termes de la liste des options.

2. Faites glisser les conditions ainsi l'ordre des cartes l'ordre des colonnes de vos données.

3. Remplissez le nom du fichier de vos données (en haut à droite) et fournir les paramètres des fichiers (à droite ci-contre).

4. Inclure le nom du fichier de toute documentation autre ensemble de données que vous pourriez inclure dans votre archive.

5. Ajouter des valeurs mondiales qui rend vos données plus complètes et plus faciles à être partagés avec d'autres utilisateurs.

6. Allez à l'onglet "meta.xml" et enregistrez le fichier XML résultant dans un dossier avec vos fichiers de données.

7. Zip le dossier en une seule archive et vous êtes faits!
